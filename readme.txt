BiblioDAM Connect allows seamless integration of BiblioDAM media onto your WordPress website(s).

== Description ==

BiblioDAM is a powerful, access controlled, cloud search media library that allows you to centrally store, control and manage your digital assets in the cloud.

BiblioDAM Connect allows seamless integration of BiblioDAM media onto your WordPress website(s).

Easily import media-rich articles as WordPress draft posts.

Media only? No problem! Easily embed BiblioDAM images and video into your posts. Classic Editor or Gutenberg are supported. For embedding videos in your branded video player, install the [Imigino Video Connect](https://wordpress.org/plugins/imigino-video-connect/ "Imigino Video Connect") plugin.

To learn more about BiblioDam, [click here](https://www.publisherstoolbox.com/bibliodam/ "Imigino Video Connect").

== Installation ==

To install, search for 'BiblioDAM Connect' under plugins, download and activate.

On the plugin settings page, setup the field values provided to you by BiblioDAM, save the settings. That's it!

For support queries please get in touch via our product website at [BiblioDAM](https://www.publisherstoolbox.com/bibliodam "BiblioDAM").

== Frequently Asked Questions ==

= Does BiblioDAM Connect work without using Imigino Video Connect? =

Yes it does.

However, you wont be able to use the branded video player. You will be able to import articles and embed images.

= Why aren't my articles importing? =

Please check your settings and make sure that you inserted the correct details that where provided to you. If you are still having issues, please [contact](https://www.publisherstoolbox.com/bibliodam/ "Imigino Video Connect") us.

= When BiblioDAM window opens, there is no content. =

Please click on the search button, the search field can be empty. This will init the load of all your content.
Some out of date browsers have this issue.
